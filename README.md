# NewFileTitle

> Add title to a new file for vim plug

## installation

* vim-plug

  ```bash
  Plug 'sunowsir/NewFileTitle'
  ```

## Usage

* Set up author : `let g:NFT_author	= "sunow"`
* Set up Mail : `let g:NFT_Mail = "sunow@139.com"`
